The XP2012 mobile app is a GNstudio.com project built using Apache cordova (formerly phonegap) and the build services offered for iOS, Android and Windows Phone.

PhoneGap is an HTML5 app platform that allows you to author native applications with web technologies and get access to APIs and app stores. PhoneGap leverages web technologies developers already know best... HTML and JavaScript. PhoneGap is an open source implementation of open standards. That means developers and companies can use PhoneGap for mobile applications that are free, commercial, open source, or any combination of these.

This app has been ideated by Giorgio Natili, desinged by Giuseppe Ansalone and coded by Tonio Di Carlo (html/css) and Giorgio Natili (jQuery mobile and javascript). The product owner of this project is Charlie Poole, many thanks also to Erik Lundh to make it happening!
